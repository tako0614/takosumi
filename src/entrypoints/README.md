# @takosjp/takosumi entrypoints

Compatibility entrypoints for the single `@takosjp/takosumi` package.

Space exports:

- `@takosjp/takosumi/contract`
- `@takosjp/takosumi/installer`
- `@takosjp/takosumi`
- `@takosjp/takosumi/runtime-agent`
- `@takosjp/takosumi/cli`
- `@takosjp/takosumi/server`

The public v1 contract is manifestless Source / Installation / Deployment / PlatformService. Operator distributions own
PlatformService inventory, provider credentials, and Terraform/OpenTofu state.
