# Managed Offering Customer Boundary {#managed-offering-customer-boundary}

This is an operational overlay for a managed Takos offering on top of Takosumi. It is not the base Cloud compatibility contract.

## Boundary

The managed offering can add customer-facing signup, terms, support, evidence, and product-profile routes. Those routes do not become Takosumi core surfaces and do not change the Cloud base profile.

## Base Cloud Surfaces

- Accounts / OIDC
- BillingPort
- Installation projection ledger
- launch token
- deploy facade
- export/import

## Managed Overlay Surfaces

- public signup gate
- terms acceptance evidence
- managed Takos launch route
- customer support / readiness evidence
- operational launch checklist

The base Cloud profile remains replaceable without adopting the managed Takos product overlay.
