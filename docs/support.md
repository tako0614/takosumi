# Support

Takosumi Cloud の課金、アカウント、利用停止、返金、障害、セキュリティ、abuse
に関する問い合わせは、次のメールアドレスに送ってください。

```text
support@takosumi.com
```

問い合わせには、可能な範囲で次の情報を含めてください。

- Takosumi Cloud に登録したメールアドレス
- Workspace / Project / Capsule / Run ID、または対象 resource 名
- 支払いに関する問い合わせの場合は、請求日、金額、Stripe receipt / invoice ID
- 問題が起きた日時と、期待した動作

Secret、provider token、private key、API key の値はメールに書かないでください。
必要な場合は Takosumi Cloud の管理画面、または support が案内する安全な手順で共有します。

Takosumi Cloud はデジタルサービスです。購入後はアカウントに plan、credit、または
managed resource access が反映されます。配送物はありません。
