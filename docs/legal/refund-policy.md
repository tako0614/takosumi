# Takosumi Cloud Refund Policy

Takosumi Cloud はデジタルサービスです。物理商品の発送、返品、配送はありません。
購入後、アカウントに plan、credit、または managed resource access が反映されます。

## 基本方針

Takosumi Cloud の月額 plan と追加 credit pack は、購入前に価格と内容を表示します。
購入後に消費済みとなった usage credit、実行済みの managed resource usage、または
提供済みの期間に対応する料金は、原則として返金対象ではありません。

ただし、次の場合は返金または調整を確認します。

- 重複決済
- 明らかな誤請求
- Takosumi Cloud 側の障害により購入した service を利用できなかった場合
- 法令上返金が必要な場合
- support が個別に返金または credit 調整が妥当と判断した場合

無料枠、trial credit、promotional credit は現金価値を持たず、返金・換金・譲渡できません。
Takosumi Cloud の credit は USD-denominated usage balance であり、現金口座ではありません。

## 返金の依頼方法

返金確認が必要な場合は、できるだけ対象決済から 14 日以内に
`support@takosumi.com` へ連絡してください。

メールには次の情報を含めてください。

- Takosumi Cloud に登録したメールアドレス
- 請求日と金額
- Stripe receipt / invoice ID がある場合はその ID
- 返金を希望する理由

Takosumi Cloud は request、usage ledger、billing record、Stripe record を確認し、
返金、credit 調整、または返金不可の理由を回答します。

## Chargebacks

不明な請求がある場合は、カード会社へ dispute する前に
`support@takosumi.com` へ連絡してください。カード明細には原則として
`TAKOSUMI` と表示されます。
