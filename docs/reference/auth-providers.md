# Auth Providers

This page is retained as a link-compatible migration stub.

Auth providers, upstream IdP brokering, passkeys, OIDC discovery, and pairwise
subject derivation are Takosumi Accounts responsibilities. The takosumi kernel
does not own OAuth/OIDC provider behavior and does not broker user identity.
