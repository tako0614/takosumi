# Retained provider artifact transparency trust

This directory retains the former public trust authority only to verify
historical evidence. The Takosumi provider artifact lane is discontinued and
is independent from Takoform publisher policy.

`provider-publisher-policy.json` describes the removed workflow that produced
former candidate evidence; it no longer admits a publication. The digest-pinned
Sigstore TrustedRoot remains available for offline verification. No existing
provider evidence is re-signed or replaced.
